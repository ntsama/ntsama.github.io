# Pack ZIP — Charte graphique IA_4_ZEP

Ce dossier contient l’ensemble des éléments visuels officiels du projet IA_4_ZEP.  
Il est conçu pour être compressé en un fichier ZIP afin de faciliter le partage, l’archivage et l’intégration dans les projets numériques.

---

## 1. Contenu du pack

Le pack ZIP doit contenir les éléments suivants :

IA_4_ZEP_Charte/
│
├── Logos/
│   ├── IA4ZEP_logo_gold.png
│   ├── IA4ZEP_logo_white.png
│   └── IA4ZEP_logo_dark.png
│
├── Banners/
│   ├── Facebook/
│   │   └── IA4ZEP_Facebook_Cover.png
│   ├── YouTube/
│   │   └── IA4ZEP_YouTube_Banner.png
│   └── LinkedIn/
│       └── IA4ZEP_LinkedIn_Banner.png
│
└── Colors_Typography/
    ├── Palette.png
    └── Typographies.pdf

---

## 2. Instructions pour créer le fichier ZIP

1. Créer un dossier nommé IA_4_ZEP_Charte
2. Ajouter les sous-dossiers :
   - Logos
   - Banners/Facebook
   - Banners/YouTube
   - Banners/LinkedIn
   - Colors_Typography
3. Placer chaque fichier dans son dossier correspondant
4. Sélectionner le dossier IA_4_ZEP_Charte
5. Choisir « Compresser » ou « Créer une archive ZIP »
6. Le fichier généré doit s’appeler :
   IA_4_ZEP_Charte.zip

---

## 3. Instructions pour importer le ZIP dans GitHub

1. Ouvrir le dépôt GitHub
2. Cliquer sur « Add file »
3. Choisir « Upload files »
4. Glisser-déposer le fichier IA_4_ZEP_Charte.zip
5. Ajouter un message de commit :
   Upload IA_4_ZEP_Charte ZIP
6. Valider le commit

---

Le pack ZIP peut être utilisé pour :
- la communication institutionnelle
- les présentations académiques
