# Theory  
## The Cognitive Trapeze & Pedagogical Swing Model  
*(English & French versions)*

---

# 🇬🇧 English Version

## 1. Theoretical Foundations  
My theoretical work explores the dynamic relationship between **language, thought, and technological mediation**.  
It builds on insights from:

- philosophy of language  
- cognitive science  
- linguistics  
- AI‑augmented learning  
- digital literacies  
- neurodiversity studies  

The central idea is that cognition is not static: it evolves through **movement**, **interaction**, and **mediation**.

---

## 2. The Cognitive Trapeze Model  
The **Cognitive Trapeze** is a dynamic model describing how learners move between:

- linguistic structures  
- cognitive representations  
- conceptual anchors  
- technological tools  

The model conceptualizes learning as a **trajectory**, not a linear process.  
Learners swing between stable points (concepts, schemas, prior knowledge) and exploratory movements (hypotheses, interpretations, problem‑solving).

This framework explains how AI tools influence:

- conceptual development  
- reasoning strategies  
- metacognitive awareness  
- multimodal understanding  

---

## 3. The Pedagogical Swing  
The **Pedagogical Swing** extends the Cognitive Trapeze to educational contexts.  
It describes how teachers and learners co‑construct meaning through:

- scaffolding  
- multimodal explanations  
- adaptive feedback  
- collaborative reasoning  
- AI‑supported mediation  

The swing represents the **rhythm of learning**:  
alternating between stability (understanding) and exploration (discovery).

---

## 4. Contribution to Cognitive Science  
The model provides:

- a unified framework for language–thought dynamics  
- a way to analyze AI‑mediated learning  
- a tool for studying multilingual and neurodiverse cognition  
- a bridge between theoretical and applied research  

It addresses a gap in current research:  
**the lack of integrated models capturing how language and thought co‑evolve in AI‑mediated environments.**

---

## 5. Relevance to My MSCA Project  
The Cognitive Trapeze is the theoretical backbone of my MSCA project **Cognitive Trapeze: Modeling Language–Thought Dynamics in AI‑Augmented Learning**.

It supports:

- conceptual analysis  
- cognitive and linguistic experiments  
- computational modeling  
- multilingual data collection  

This theory provides the **conceptual architecture** for understanding how AI reshapes learning trajectories.

# 🇫🇷 Version Française

## 1. Fondements Théoriques  
Mes travaux théoriques explorent la relation dynamique entre **langage, pensée et médiation technologique**.  
Ils s’appuient sur :

- la philosophie du langage  
- les sciences cognitives  
- la linguistique  
- l’apprentissage augmenté par IA  
- les littératies numériques  
- les études sur la neurodiversité  

L’idée centrale : la cognition est un **mouvement**, pas un état fixe.

---

## 2. Le Modèle du Trapèze Cognitif  
Le **Trapèze Cognitif** décrit la manière dont les apprenants se déplacent entre :

- structures linguistiques  
- représentations cognitives  
- points d’appui conceptuels  
- outils technologiques  

L’apprentissage est vu comme une **trajectoire dynamique**, alternant stabilité et exploration.

Ce modèle permet d’analyser l’influence de l’IA sur :

- le développement conceptuel  
- les stratégies de raisonnement  
- la conscience métacognitive  
- la compréhension multimodale  

---

## 3. Le Balançoire Pédagogique  
La **Balançoire Pédagogique** étend le Trapèze Cognitif au domaine éducatif.  
Elle décrit la co‑construction du sens entre enseignants et apprenants à travers :

- l’étayage  
- les explications multimodales  
- le feedback adaptatif  
- le raisonnement collaboratif  
- la médiation assistée par IA  

La balançoire représente le **rythme de l’apprentissage** :  
alternance entre stabilité (comprendre) et exploration (découvrir).

---

## 4. Contribution aux Sciences Cognitives  
Le modèle propose :

- un cadre unifié pour les dynamiques langage–pensée  
- une analyse de l’apprentissage médié par IA  
- un outil pour étudier les profils multilingues et neurodivergents  
- un pont entre théorie et pratique  

Il répond à un manque :  
**l’absence de modèles intégrés capturant la co‑évolution langage–pensée dans des environnements médiés par la technologie.**

---

## 5. Lien avec Mon Projet MSCA  
Le Trapèze Cognitif constitue l’ossature théorique de mon projet MSCA **Cognitive Trapeze**.

Il soutient :

- l’analyse conceptuelle  
- les expériences cognitives et linguistiques  
- la modélisation computationnelle  
- la collecte de données multilingues  

Cette théorie fournit l’architecture conceptuelle pour comprendre comment l’IA transforme les trajectoires d’apprentissage.
